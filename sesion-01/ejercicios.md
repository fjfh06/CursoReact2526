# Ejercicios

## Ejercicio 4:

Crear un componente que gestiones una lista de contactos el usuario puede agregar y eliminar contactos para ello los requisitos son:
1º Crear una interfaz llamada contacto con el id, nombre, telefono, email
2º El componente debe de tener un formulario con 3 inputs nombre telefono email 
3º Un boton para agregar contacto
4º Una lista para mostrar contactos
5º Un boton para eliminar contacto


Las funcionalidades son:
- Agregar un nuevo contacto (generar un id automatico con akguna libreria npm)
- Eliminar contacto por su id
- Limpiar el form despues de agregar
- Mostrar un mensaje sino hay contactos almacenados
